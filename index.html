📘 Les commandes essentielles

Action : Parier

!matchs

Affiche le calendrier numéroté des matchs à venir. Utilise le numéro pour parier ou défier.

!pari 3 2-1

Pari sur le match n°3. Score exact = 3 pts, bon résultat = 1 pt.

!duel @Ami 3 20

Défie un pote sur le match n°3 avec une mise de 20 pts. Le gagnant prend tout.

Action : Classements

!classement

Classement général (joueurs non-VIP). Points accumulés sur la saison.

!classement VIP

Ligue Elite — seuls les points marqués en étant abonné VIP comptent.

!profil

Tes statistiques complètes : grade, winrate, points et palmarès.
